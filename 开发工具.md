### IntelliJ
**下载地址**：[IntelliJ 官方下载](https://www.jetbrains.com/zh-cn/idea/download/?section=windows)
#### 调整启动配置
创建或编辑 `idea.properties` 文件，在文件中添加如下内容：
```properties
# idea.properties实例

# 系统缓存（索引、编译缓存）
idea.system.path=D:/Dev/Editor/config/system/IntelliJ

# 配置目录（快捷键、UI、主题、IDE 设置）
idea.config.path=D:/Dev/Editor/config/config/IntelliJ

# 插件目录
idea.plugins.path=D:/Dev/Editor/config/plugins/IntelliJ

# 日志目录
idea.log.path=D:/Dev/Editor/config/log/IntelliJ
```

## Git
**下载地址**：[Git 官方下载](https://git-scm.com/install/windows)
### 相关操作
```shell
# 设置全局用户名
git config --global user.name "你的名字"

# 设置全局邮箱
git config --global user.email "your.email@example.com"

# 设置代理
git config --global http.proxy http://127.0.0.1:10808
git config --global https.proxy http://127.0.0.1:10808
```
