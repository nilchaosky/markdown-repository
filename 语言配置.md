# Golang
**下载地址**：[Golang 官方下载](https://go.dev/dl/)
## 环境配置
window配置
```
# 新建系统变量GOROOT
变量名：GOROOT
变量值：go安装路径
# 将bin目录添加到系统变量PATH
%GOROOT%\bin

# 修改用户变量GOPATH
变量名：GOPATH
变量值：{自定义go工作目录}
# 修改用户变量PATH
将默认%USERPROFILE%\go\bin修改成%GOPATH%\bin
```

go 配置
```shell
# 设置国内代理
go env -w GOPROXY=https://goproxy.cn,direct
# 开启 Go Modules 作为依赖管理机制
go env -w GO111MODULE=on
# 全局关闭 CGO，强制使用纯 Go 编译方式
go env -w CGO_ENABLED=0
```
# Node.js
## NVM
**下载地址**：[NVM GitHub下载](https://github.com/coreybutler/nvm-windows/releases)
### 相关操作
```shell
# 查看版本
nvm -v
# 显示可以安装的所有node.js的版本
nvm list available
# 安装版本
nvm install <node_version>
# 查看已安装的node.js
nvm list
# 切换到使用指定的nodejs版本
nvm use <node_version>
# 卸载
nvm uninstall <node_version>
```
npm配置
```shell
# 设置全局安装包目录
npm config set cache D:\Dev\PL\npm\npm-global --global
# 设置全局npm 缓存
npm config set cache D:\Dev\PL\npm\npm-cache --global
# 设置淘宝镜像
npm config set registry https://registry.npmmirror.com/
```
